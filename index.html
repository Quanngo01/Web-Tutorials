Hướng dẫn sử dụng Git log

  • "git branch -M ..." : change branch's name
  • "git branch -c ..." : creat branch
  • "git branch -d||-D ..." : delete another local branch
  • "git push -d origin ..." : delete remote branch
  • "git checkout ..." : switch branch
  • "git checkout -b ..." : creat and switch branch
  • "git branch (-a||--all)" : show all branch
  • "git log --oneline" : show all commit (ID + message)
  • "git pull" : pull request
  • "git clone ..." : clone a repository
  • "git push -u||--set-upstream origin ..." : push a new branch
